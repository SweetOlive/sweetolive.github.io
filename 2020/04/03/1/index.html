Hexo博客在换了电脑或者重装系统后如何在新电脑恢复

发表于 2020-04-03 更新于 2022-09-20 分类于 Hexo 阅读次数：
本文字数： 894 阅读时长 ≈ 1 分钟

因为过年换了新电脑，然后因为疫情的事情，把博客给忘了，现在抽空重新把博客环境在新电脑上面搭建起来，于是写了这篇分享。

在新的环境下，自行确保安装了Node.js、Git

一般情况下，新电脑新系统都是没有SSH Key，要重新配置

打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容,打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：
